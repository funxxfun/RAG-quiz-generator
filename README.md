# 🧠 RAGクイズジェネレーター

WebページのテキストをAIに読み込ませて、オリジナルクイズを自動生成する学習アプリです。

---

## 📖 概要

任意のWebページの内容をコピー＆ペーストするだけで、Claude AIが**まるばつ・4択・穴埋め**の3形式のクイズを自動生成します。技術ブログや記事を効率よく学習するのに最適です。

---

## 🚀 使い方

### Step 1 - テキストを取り込む
1. 学習したいWebページをブラウザで開く
2. 記事本文を選択して `Ctrl+C` でコピー
3. アプリのテキストエリアに `Ctrl+V` で貼り付け
4. 「追加する」ボタンをクリック
5. 複数の記事を繰り返し追加することも可能

### Step 2 - クイズを生成する
1. 問題数を選択（5 / 10 / 15 / 20問）
2. 「🤖 クイズを生成する」ボタンをクリック
3. Claude AIが内容を分析してクイズを自動作成

### Step 3 - クイズで学習する
1. 「▶ クイズスタート」をクリック
2. まるばつ・4択・穴埋めの問題に答える
3. 正解・不正解の結果と解説を確認
4. 最後にスコアを確認

---

## ✨ 機能

| 機能 | 説明 |
|------|------|
| テキスト取り込み | コピペで任意のWebページを学習素材に |
| 複数ソース対応 | 複数記事を重ねて豊富なクイズを生成 |
| 3種類の問題形式 | まるばつ・4択・穴埋めをバランスよく出題 |
| 解説付き | 正解・不正解どちらにも解説（ヒント）を表示 |
| スコア記録 | セッション中の正解数をリアルタイム表示 |

---

## 🛠 技術スタック

- **フロントエンド**: React (TypeScript)
- **AI**: Claude API (`claude-sonnet-4-20250514`)
- **スタイリング**: インラインスタイル（Tailwind不使用）
- **動作環境**: Claude.ai アーティファクト

---

## ⚙️ クイズ生成ロジック

取り込んだテキスト（最大6,000文字/ソース、合計8,000文字）をClaude APIに渡し、以下の比率でJSON形式のクイズを生成します。

- まるばつ：約40%
- 4択：約40%
- 穴埋め：約20%

---

## ⚠️ 注意事項

- Claude.aiのアーティファクト環境ではURLからの直接フェッチが動作しないため、**テキストの手動貼り付けのみ**対応しています
- 1ソースあたり最大6,000文字まで取り込み可能です
- クイズのスコアはセッション中のみ保持されます（ページ更新でリセット）
